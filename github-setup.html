Getting Started with GitHub & VS Code

A step-by-step guide for Gonzaga students to set up their development environment with GitHub and GitHub Copilot.

1. Create Your GitHub Account üöÄ

  1. Go to github.com/signup
  2. Click "Sign up"
  3. Use your @gonzaga.org email address
  4. Choose a unique username
  5. Create a strong password
  6. Verify your email address when prompted

2. Get GitHub Student Benefits & Copilot üéì

Set up two-factor authentication (2FA) - Required for Copilot:

  1. Go to your GitHub profile settings
  2. Click "Password and authentication"
  3. Under "Two-factor authentication," click "Enable 2FA"
  4. Follow the prompts to set up using an authenticator app (recommended) or SMS

Add your address information - Required for Copilot:

  1. Go to your GitHub profile settings
  2. Click "Billing and plans"
  3. Click "Add billing address"
  4. Fill in your information (for students, your school address is fine)
  5. Save your information

Apply for GitHub Student Benefits:

  1. Visit education.github.com/benefits
  2. Click "Get student benefits"
  3. Sign in with your GitHub account
  4. Select "Student"
  5. Use your @gonzaga.org email
  6. For proof of student status:
    1. Go to your Gonzaga profile page: gonzaga.myschoolapp.com
    2. Take a full-page screenshot:
      • Windows: Press Windows+Shift+S, then select "Full-screen"
      • Mac: Press Command+Shift+3
    3. Upload this screenshot when prompted by GitHub
  7. Wait for verification email (usually within 48 hours)
  8. Once approved, activate GitHub Copilot from your benefits page

3. Set Up VS Code with GitHub üíª

Download & install Git:

For Windows Users:

  1. Go to git-scm.com/download/win
  2. Download the installer and follow the prompts (use default settings)

For Mac Users:

  1. Restart your computer
  2. Install Homebrew first (if not already installed):
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

    Follow the prompts and wait for the terminal to install Xcode completely (may take a few minutes)

    After installation, Homebrew will show you some commands to run. They will look something like this:

    ==> Next steps: - Run these two commands in your terminal to add Homebrew to your PATH: (echo; echo 'eval "$(/opt/homebrew/bin/brew shellenv)"') >> ~/.zprofile eval "$(/opt/homebrew/bin/brew shellenv)"

    Run these commands exactly as shown in your terminal. This ensures Homebrew is properly added to your system's PATH.

  3. Then install Git using Homebrew:
    brew install git

Verify Git Installation:

git --version

Set Up VS Code:

  1. Download & install VS Code from code.visualstudio.com
  2. Install these VS Code extensions:
    • GitHub Copilot
  3. Sign into GitHub in VS Code:
    1. Click the profile icon in the bottom left
    2. Select "Account and Sync Settings"
    3. Choose "Sign in"
    4. Choose "Sign in with GitHub"
    5. Complete authentication in your browser

4. Create & Publish Your First Site üåê

Create a new project folder:

  1. Open VS Code
  2. Click on the "Explorer" tab in the left sidebar (or press Ctrl+Shift+E / Cmd+Shift+E)
  3. Click "Open Folder" button
  4. Select a location on your computer
  5. Click "New Folder" button
  6. Name your folder (e.g., "my-first-website")
  7. Click "Select Folder" to open it

Create an index.html file:

  1. In the Explorer panel, hover over your folder name
  2. Click the "New File" icon (looks like a file with a + sign)
  3. Type "index.html" and press Enter
  4. The new file will open in the editor

Add the template code:

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Bootstrap demo</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-T3c6CoIi6uLrA9TneNEoa7RxnatzjcDSCmG1MXxSR1GAsXEV/Dwwykc2MPK8M2HN" crossorigin="anonymous">
    <link rel="stylesheet" href="styles.css">
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script type="text/javascript" src="script.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-C6RzsynM9kWDrMNeT87bh95OGNyZPhcTNXj1NW7RuBCsyN/o0jlpcV8Qyq46cDfL" crossorigin="anonymous"></script>
  </body>
</html>

Publish to GitHub:

  1. Click the "Source Control" tab in VS Code (branch icon)
  2. Click "Publish to GitHub"
  3. Choose "Public Repository"
  4. Select your folder
  5. Add a commit message
  6. Click "Publish"

Enable GitHub Pages:

  1. Go to your repository on GitHub.com
  2. Click "Settings"
  3. Click "Pages" in the left sidebar
  4. Under "Branch", select "main" or "master"
  5. Click "Save"
  6. Wait a few minutes - your site will be live at: https://[your-username].github.io/[repository-name]

Need Help? üÜò

  • Ask your teacher for assistance
  • Check error messages carefully
  • Visit docs.github.com for documentation
  • Use the VS Code GitHub extension for visual Git controls

Remember: Always commit and sync your changes to save your work online!
